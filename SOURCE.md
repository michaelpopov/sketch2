# Source Index

- `.clang-format`: Formatting rules for the C/C++ source tree.
- `.gitignore`: Git ignore patterns for build outputs and local artifacts.
- `CMakeLists.txt`: Top-level CMake entry point for configuring the project.
- `Makefile`: Convenience wrapper around common build and test commands.
- `README.md`: Main project overview and usage notes.
- `commands.txt`: Scratch list of shell commands used during development.
