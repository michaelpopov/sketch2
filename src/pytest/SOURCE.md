# Source Index

- `demo.py`: Small Python demo script for exercising the parasol bindings with text or binary dataset loading paths.
- `parasol_wrapper.py`: Python wrapper layer around the parasol shared library, including bulk text and binary generation helpers.
- `shell.py`: Interactive helper shell for running parasol operations from Python.
- `test_parasol_basic.py`: Basic Python tests for the parasol wrapper.
- `test_parasol_errors.py`: Python tests covering parasol error handling.
