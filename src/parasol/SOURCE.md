# Source Index

- `CMakeLists.txt`: Build rules for the parasol library and tests.
- `DESIGN.md`: Design notes for the parasol API layer.
- `README.md`: Overview of the parasol C API and usage model.
- `parasol.cpp`: C API implementation for dataset lifecycle, updates, queries, and text/binary bulk generation or loading.
- `parasol.h`: Public C API declarations exposed by parasol, including text and binary bulk-ingest helpers.
- `utest_main.cpp`: GoogleTest entry point for the parasol test binary.
- `utest_parasol.cpp`: Unit tests for the parasol API.
